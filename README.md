# Anichin Lite

Anichin Lite adalah aplikasi Android berbasis WebView yang dirancang untuk memberikan pengalaman menjelajah situs anichin.moe dengan tampilan yang lebih sederhana, ringan, dan nyaman digunakan.

Aplikasi ini berfokus pada peningkatan pengalaman pengguna melalui antarmuka yang lebih bersih, navigasi yang praktis, serta berbagai optimasi agar penggunaan terasa lebih lancar di perangkat Android.

## Fitur

**Tampilan Lebih Bersih**  
Menyederhanakan tampilan halaman dengan menyembunyikan elemen yang tidak diperlukan sehingga konten utama lebih mudah dilihat.

**Pengalaman Browsing Lebih Nyaman**  
Mengurangi gangguan dari elemen yang dapat menghambat kenyamanan saat membuka halaman.

**Anti Redirect**  
Membantu mencegah pengalihan otomatis ke halaman atau domain yang tidak diinginkan.

**Pemilihan Server Otomatis**  
Jika tersedia, aplikasi akan memilih server OK.ru secara otomatis pada halaman episode.

**Fullscreen Video**  
Mendukung pemutaran video dalam mode layar penuh.

**Landscape Otomatis**  
Layar akan beralih ke mode landscape secara otomatis saat video memasuki mode fullscreen.

**Mode Imersif**  
Status bar dan navigation bar disembunyikan saat fullscreen sehingga area tampilan video menjadi lebih luas.

**Layar Tetap Menyala**  
Mencegah layar mati secara otomatis selama video diputar dalam mode fullscreen.

**Dukungan Layar Modern**  
Mendukung tampilan layar penuh pada perangkat dengan notch maupun punch-hole.

**Pull-to-Refresh**  
Tarik layar ke bawah untuk memuat ulang halaman dengan cepat.

**Indikator Refresh**  
Menampilkan indikator sederhana saat halaman sedang diperbarui.

**Indikator Loading Halaman**  
Menampilkan progress loading di bagian atas layar ketika halaman sedang dimuat.

**Halaman Error Khusus**  
Menampilkan halaman "Koneksi Terputus" beserta tombol Coba Lagi apabila terjadi masalah jaringan.

**Riwayat dan Bookmark Tersimpan**  
Riwayat penjelajahan dan bookmark tetap tersimpan melalui penyimpanan WebView selama data aplikasi tidak dihapus.
